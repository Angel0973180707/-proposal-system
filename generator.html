品牌成交系統 · 提案產生器

填一次資料，產生一份客戶專屬互動提案

把不同客戶的品牌內容填入下方欄位，右下角按「產生提案網頁」即可下載一份完整可用的 HTML 提案檔，架構與互動功能與樣板完全一致。

📋 開始前，先看這裡：要準備什麼資料？產生後怎麼用？（點擊展開）
① 要填什麼資料
前置作業建議先跟客戶開一次簡短訪談或看過去的品牌素材，準備好：品牌一句話定位、3種語氣不同的主標語、30秒／60秒品牌介紹文案、品牌可用的3種信任角色（如創辦人/團隊/專家）、智慧名片想保留的功能區塊、經營資源可投入的方向、內容規劃的分類與占比。沒有的欄位可以先填通用預設值，之後跟客戶開會時當場調整即可，不用一次到位。

② 文案怎麼回填給客戶
填完表單按「產生提案網頁」，會下載一份獨立的 .html 檔案，就是最終要給客戶看的東西。這個檔案跟這個「產生器」互相獨立——之後不管是否還留著產生器裡的資料，這份下載的檔案本身永遠完整可用。

③ 怎麼交付給客戶
不建議直接傳 .html 檔案讓客戶用手機「檔案檢視器」打開（互動按鈕不會動）。建議二選一：
· 上傳到您慣用的 GitHub Pages，取得網址後用 LINE 傳連結給客戶，點開即可用手機瀏覽器操作
· 提案會議現場用您自己的電腦/平板開啟這個檔案，帶著客戶一起互動勾選

④ 客戶填完之後資料在哪裡
若已設定「GAS 回傳網址」，客戶按下「產生我的提案摘要」時資料會自動送進您的 Google Sheets；若沒設定，客戶畫面上仍會顯示摘要，需要客戶手動複製傳給您。

快速

兩步驟：跟 Claude 對話產出內容 → 貼回這裡套用

Step 1｜複製指令，去跟 Claude 對話

按下方按鈕複製一段指令，貼到任何一個 Claude 對話視窗，把您手上的品牌素材也一起貼給它。Claude 會直接幫您整理成這個工具看得懂的 JSON 格式。

Step 2｜把 Claude 的完整回覆貼回這裡

不用自己挑內容，Claude 回覆的整段文字（含前後說明文字都沒關係）直接全部複製貼進下面這格，按「套用到表單」。這一步是純本地解析，不會呼叫任何外部服務，在哪裡打開這個檔案都能用。

提醒：套用到表單只是把內容填進下面的欄位，還沒有存檔。填完後仍需要往下捲到最底部按「⚡ 一鍵產生提案並回填試算表」，才會真正下載提案檔並存進您的 Google Sheets。
